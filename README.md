## About The Project

### Features

-   Clean & Organized Code.
-   Fully Responsive Design
-   Moderator Panel Login System.
-   Moderator Panel With CRUD Operations for everyting on the website.
-   Fully Working Contact and Report Forms, Storing In Database.

## Getting Started

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/Astennu-Corporation/mzd.git
    ```
2. Composer Install
    ```sh
    composer install
    ```
3. Generate Key
    ```sh
    php artisan key:generate
    ```
4. Run Migrations
    ```sh
    php artisan migrate
    ```
5. Seed The Database
    ```sh
    php artisan db:seed
    ```
6. Start The Server
    ```sh
    php artisan serve
    ```
